# v3 Risk Estimator

Interactive risk estimator that predicts the probability of a country being in a high-risk suicide category.

## Outputs
- `v3/data_clean/v3_features_v1.csv`
- `v3/data_clean/v3_features_v2.csv`
- `v3/report/v3_feature_summary.md`
- `v3/notebooks`: feature + modeling notebooks

## Rebuild
```bash
python scripts/run_v3_pipeline.py
```

## Run
```bash
python scripts/run_app.py --version v3
```

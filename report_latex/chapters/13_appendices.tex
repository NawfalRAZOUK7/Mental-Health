\chapter{Annexes}
\section{Catalogue des datasets}
Le catalogue complet est fourni dans \texttt{v1/data\_clean/dataset\_catalog.csv}. Il liste chaque fichier, ses colonnes et ses valeurs uniques par dimension (année, sexe, âge, cause, mesure, métrique).

\section{Matrice d'usage}
La matrice d'usage est disponible dans \texttt{v1/report/dataset\_usage\_matrix.md}. Elle justifie l'utilisation de chaque dataset dans les pages du tableau de bord et dans la table ML.

\section{Rapport de qualité de fusion}
Le rapport de fusion est disponible dans \texttt{v1/report/merge\_quality.md}. Il documente les tailles avant/après, les filtres appliqués et la note de fusion inter-années.
